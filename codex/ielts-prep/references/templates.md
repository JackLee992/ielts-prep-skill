# Templates

Use these templates when producing recurring IELTS coaching outputs.

## 30-Minute Study Block

```markdown
### 30-Minute Block

**Focus**: [single bottleneck]
**Goal**: [what should improve by the end]

1. Warm-up (5 min)
- [tiny activation task]

2. Main drill (15 min)
- [focused item set or speaking/writing drill]

3. Review (7 min)
- [classify mistakes]
- [write one rule]

4. Reset (3 min)
- [re-answer one item or rewrite one sentence]

**Next drill**: [what to do next]
**Re-test**: [when and how]
```

## 7-Day Plan

```markdown
### 7-Day IELTS Plan

**Primary module**: [Listening/Reading/Writing/Speaking]
**Support module**: [secondary focus]
**Daily study time**: [x minutes]

Day 1: [diagnosis + short drill]
Day 2: [focused drill]
Day 3: [review + correction]
Day 4: [focused drill]
Day 5: [timed mixed practice]
Day 6: [speaking or writing output]
Day 7: [review + reset + next-week adjustment]

**Main rule for the week**: [single behavior rule]
**Retest point**: [specific task on day 6 or 7]
```
