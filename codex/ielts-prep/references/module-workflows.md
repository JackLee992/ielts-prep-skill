# Module Workflows

Use this file when the user needs section-specific diagnosis, drill design, or review logic.

## Listening

### Diagnose by failure type

- Audio processing: misses linked speech, stress shifts, reduced forms
- Distractor handling: changes answer too early after hearing a familiar word
- Prediction weakness: starts listening without anticipating word class, number, or topic
- Form accuracy: spelling, plurals, capitalization, dates, or numbers
- Map and label execution: loses position after a direction change

### High-yield drills

- Shadow short audio segments and mark stressed words
- Predict answer type before audio starts
- Re-listen only to distractor-heavy questions and explain the trap
- Drill spelling and number transcription under time pressure
- Practice map tasks by pausing before each turn and verbalizing location logic

### Review prompts

- What exact signal was missed?
- Was the answer heard but not trusted?
- Did the mistake come from comprehension, memory, or form?

## Reading

### Diagnose by failure type

- Timing: spends too long locating evidence
- Paraphrase weakness: fails to connect passage wording to question wording
- Question-type weakness: weak strategy on TFNG, Matching Headings, Sentence Completion, or Matching Information
- Over-inference: answers from world knowledge or impression instead of passage evidence
- Precision weakness: ignores qualifiers such as `some`, `mainly`, `only`, `often`, `less likely`

### High-yield drills

- Limit passage scanning to 60 to 90 seconds and mark anchor nouns
- Complete paraphrase matching sets without answering the questions first
- Do short TFNG sets and justify each answer with one evidence sentence
- Mark qualifiers before locating evidence
- Rework only the wrong questions and classify the mistake type

### Review prompts

- Did the passage clearly support the answer?
- Was the problem locating, matching, or deciding?
- Which qualifier or contrast marker changed the meaning?

## Writing

### Task 1

Focus on:

- Overview quality
- Selection of key features
- Grouping and comparison logic
- Data accuracy and grammar control

High-yield drills:

- Write only the overview for three charts
- Group key features before writing any sentences
- Rewrite one body paragraph for comparison clarity

### Task 2

Focus on:

- Clear position
- Paragraph purpose
- Support depth
- Cohesion without overusing connectors
- Grammar stability under pressure

High-yield drills:

- Plan thesis plus two topic sentences in 5 minutes
- Expand one weak idea into claim, explanation, example
- Rewrite one paragraph for logic and sentence variety
- Isolate repeated grammar errors and correct them in batches

### Review prompts

- Does each paragraph do one clear job?
- Is the support specific enough to sound convincing?
- Are grammar errors random or patterned?
- Would fixing structure or language bring the faster score gain?

## Speaking

### Diagnose by failure type

- Fluency breakdown: long pauses, self-restarts, filler dependence
- Idea depth: answers too short or abstract
- Grammar instability: loses control when answers get longer
- Lexical repetition: repeats the same safe words and sentence frames
- Pronunciation clarity: unclear stress, swallowed endings, rhythm issues

### High-yield drills

- Answer one Part 1 topic with a 20-second limit per answer
- Practice Part 2 with a 1-minute plan and 2-minute response
- Extend Part 3 answers with `point -> reason -> example -> implication`
- Record short answers and mark repeated words or broken sentence patterns
- Re-answer only one weak question after feedback

### Review prompts

- Did the user run out of ideas or language?
- Which sentence pattern kept breaking?
- Would a response frame unlock fluency, or is the problem vocabulary depth?

## Mock-Test Review

Use this sequence after a mock:

1. Identify score drop by module and question type
2. Separate timing losses from language losses
3. Select one primary bottleneck and one support bottleneck
4. Assign drills for the next 3 to 5 study sessions
5. Set one retest using similar material

Do not respond to mock results with a generic "practice more" plan.
