# User Profiles

Use this file when you need to adapt coaching style, question order, and plan intensity to the user's situation.

## First-Timer

Typical traits:

- Has not taken IELTS before
- Knows general English level but not exam mechanics
- Often underestimates timing pressure

Coach this user by:

- Explaining the test structure early
- Running a broad diagnostic before deep specialization
- Teaching timing rules explicitly
- Correcting "test-naive" mistakes such as spending too long on one item

Ask first:

- Academic or General Training?
- Exam date?
- Any prior mock scores?
- Daily study time?

## Retaker

Typical traits:

- Has taken IELTS before and missed the target
- Usually knows the format already
- Often has one or two recurring bottlenecks

Coach this user by:

- Asking for the previous score breakdown first
- Comparing current problems to the prior attempt
- Focusing 70% to 80% of effort on the weakest repeat pattern
- Avoiding broad "study everything again" plans

Ask first:

- Previous band breakdown?
- Which section felt worst?
- Which score gap matters most now?
- What changed since the last attempt?

## Busy Professional

Typical traits:

- Limited weekday study time
- Often preparing for immigration or work-related goals
- Needs efficient plans and visible progress

Coach this user by:

- Designing short weekday drills
- Preserving one longer weekend block
- Prioritizing score ROI over perfect balance
- Converting big tasks into 15- to 30-minute sessions

Ask first:

- Real weekday study budget?
- Weekend availability?
- Deadline or application milestone?
- Academic or General Training?

## Professional Registration Candidate

Typical traits:

- Needs strict per-section minimums
- Often has especially high Writing or Speaking requirements
- Stakes are usually professional and urgent

Coach this user by:

- Tracking section minimums more carefully than overall band
- Stress-testing Writing and Speaking consistency
- Giving formal-register corrections when appropriate
- Warning early when the current score pattern is incompatible with the requirement

Ask first:

- Which authority or profession?
- Exact per-section minimums?
- Current strongest and weakest sections?
- Hard deadline?

## Plateaued Repeat Taker

Typical traits:

- Multiple attempts with little movement
- Same error types keep returning
- May be burned out or over-practicing strengths

Coach this user by:

- Naming the plateau directly and calmly
- Shrinking the focus to one criterion or one question type
- Replacing quantity with targeted drill-retest cycles
- Checking whether the user needs strategy change, rest, or a different test path

Ask first:

- How many attempts so far?
- What has stayed the same across attempts?
- Which error type repeats most?
- Are they open to changing strategy?

## Parent or External Supporter

Typical traits:

- Wants structure and progress visibility
- May not need deep linguistic detail
- Needs clear "on track / off track" reporting

Coach this user by:

- Summarizing progress clearly
- Turning weekly work into a simple checklist
- Highlighting risks early
- Keeping feedback readable and low-jargon

Ask first:

- Target score and deadline?
- Current mock level?
- How often should progress be reviewed?

## Routing Rule

If the user fits multiple profiles, prioritize in this order:

1. Professional requirement or immigration deadline
2. Retaker / plateau status
3. Time-budget constraint
4. First-time familiarity needs

The strongest external constraint should shape the plan.
