# Targets by Purpose

Use this file when the user asks what score they need, whether their current score is enough, or how to prioritize prep based on a real-world goal.

Always present these as **typical ranges**, not guaranteed requirements. Encourage the user to verify the latest official requirement for the specific institution, pathway, or authority.

## University Admission

Typical patterns:

- Foundation programs: around 5.5 overall
- Undergraduate admission: around 6.0 to 6.5 overall
- Postgraduate admission: around 6.5 to 7.0 overall
- Highly competitive programs: around 7.0 to 7.5 overall

Coaching implications:

- Track per-section minimums, not only overall band
- Writing and Speaking often become the bottlenecks for higher targets
- For short timelines, raise the lowest scoring section before polishing strengths

## Immigration

### Canada-style pathways

Typical pattern:

- Threshold jumps matter more than tiny score improvements
- A single section crossing a benchmark can change the total points outcome substantially

Coaching implications:

- Map prep around score thresholds, not vanity bands
- Treat each section minimum as strategic
- If one section alone blocks the target, focus there first

### Australia-style pathways

Typical pattern:

- Different score bands can create major point differences
- Strong section balance is often more valuable than one standout section

Coaching implications:

- Prioritize section consistency
- Stress-test all four modules if the user is near a key points threshold

## Professional Registration

Typical pattern:

- Exact per-section minima matter
- Overall score alone may be insufficient

Coaching implications:

- Treat any sub-threshold section as critical
- Build more conservative plans around Writing and Speaking consistency
- Use more frequent retests on the weakest required section

## General Improvement Without External Deadline

If the user is studying without a hard external requirement:

- Set one realistic overall target
- Set one support target for the weakest section
- Avoid over-optimizing for exact external thresholds

## One Skill Retake Logic

Raise this option when all are true:

- The user has already taken the test
- One section clearly underperforms the others
- That single section blocks the user's real-world goal
- Improvement of 0.5 to 1.0 band in that section seems realistic

Do not push this option when:

- Multiple sections are weak
- The user does not yet understand the exam format
- Score volatility is still high across modules
