# IELTS Error Log

Use one entry per repeated or high-value mistake.

```markdown
## [YYYY-MM-DD] [module] [question type or task]

**Mistake**: [what went wrong]
**Root cause**: [timing / language / strategy / attention / confidence]
**Evidence**: [one concrete example]
**Rule**: [what to do next time]
**Drill**: [short targeted practice]
**Retest date**: [YYYY-MM-DD]
**Status**: open | improving | stable
```

Suggested tags:

- Listening: distractor, spelling, numbers, map, prediction
- Reading: tfng, headings, matching, timing, paraphrase
- Writing: thesis, support, cohesion, grammar, task-response
- Speaking: fluency, idea-depth, grammar, lexical-range, pronunciation
